# Vim

サーバー上で作業するときは、設定ファイルや YAML をターミナル上で編集する場面があります。
このハンズオンでは `vim` の操作にも少し慣れることを目指します。

すでに `vim` が使える人は、そのまま進めてください。
`emacs`、`nano`、VS Code など、使いたいエディタがある人はそれを使って構いません。

何も分からない場合は、このページの操作だけ覚えれば大丈夫です。

## Vim を開く

ファイルを開く、または新しく作るには次のように実行します。

```bash
vim my-httpd.yaml
```

Vim には主に 2 つの状態があります。

- ノーマルモード: 移動、保存、終了などをする状態
- 入力モード: 文字を入力する状態

開いた直後はノーマルモードです。

## 文字を入力する

文字を入力したい場所に移動して、`i` を押します。
画面下に `-- INSERT --` と表示されれば、入力モードです。

```text
i
```

入力が終わったら、`Esc` を押してノーマルモードに戻ります。

```text
Esc
```

::: tip
Vim で困ったら、まず `Esc` を押してください。
ノーマルモードに戻ると、保存や終了の操作ができます。
:::

## 保存して終了する

入力が終わったら、`Esc` を押してから次のように入力します。

```text
:wq
```

最後に `Enter` を押します。

意味は次の通りです。

- `:w`: 保存する
- `:q`: 終了する
- `:wq`: 保存して終了する

## 保存せずに終了する

間違えて開いた場合や、変更を捨てたい場合は、`Esc` を押してから次のように入力します。

```text
:q!
```

最後に `Enter` を押します。

## よく使う操作

| 操作      | 意味                   |
| --------- | ---------------------- |
| `i`       | 入力モードに入る       |
| `Esc`     | ノーマルモードに戻る   |
| `:w`      | 保存する               |
| `:wq`     | 保存して終了する       |
| `:q!`     | 保存せずに終了する     |
| `/文字列` | 文字列を検索する       |
| `n`       | 次の検索結果へ移動する |
| `dd`      | 1 行削除する           |
| `u`       | 1 つ前の操作を取り消す |

## このハンズオンでの使い方

Manifest や設定ファイルを編集するときは、たとえば次のように開きます。

```bash
vim my-httpd.yaml
```

編集の流れは、基本的に次の 4 ステップです。

1. `vim ファイル名` で開く
2. `i` で入力モードに入る
3. 編集が終わったら `Esc` を押す
4. `:wq` で保存して終了する

この 4 ステップだけ分かれば、ハンズオン中の編集は進められます。
